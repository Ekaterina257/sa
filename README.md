# Калькулятор подсети

Веб-приложение для расчёта сетевых параметров. Позволяет ввести IP-адрес и маску подсети, а затем автоматически определяет:
- Адрес сети
- Количество доступных IP-адресов для подключения устройств

Интерфейс построен с использованием собственной UI-библиотеки компонентов, разработанной специально для этого проекта.

## Технологии
- **Vue 3** — современный фреймворк для создания пользовательских интерфейсов
- **TypeScript** — строгая типизация для надёжного и читаемого кода
- **SCSS** — стили с поддержкой переменных и вложенных правил
- **subnet-ui-katya** — персонализированная UI-библиотека компонентов

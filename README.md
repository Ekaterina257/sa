# Калькулятор подсети

Это веб-приложение для расчёта сетевых параметров: оно принимает IP-адрес и маску подсети, а затем показывает адрес сети и сколько устройств можно подключить в эту сеть.

## Технологии
- Vue 3
- TypeScript
- SCSS
